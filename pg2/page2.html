O que é Média?

A média aritmética, frequentemente chamada de média, é uma medida de tendência central que representa o valor médio de um conjunto de dados. Para calculá-la, você soma todos os valores do conjunto e divide o resultado pelo número total de elementos.

Fórmula: Média = (Σx) / n

Exemplo: Suponha que você tem as notas 7, 8 e 9.

Para encontrar a média, você soma os valores: 7 + 8 + 9 = 24, e divide pelo número de notas:

24 / 3 = 8. Portanto, a média das notas é 8.

O que é Mediana?

A mediana é a medida de tendência central que divide um conjunto de dados ordenados em duas partes iguais. Se o número de elementos for ímpar, a mediana é o valor central. Se for par, a mediana é a média dos dois valores centrais.

Fórmula: Mediana = valor central (se ímpar) ou (valor central inferior + valor central superior) / 2 (se par)

Exemplo: Para a sequência de números 1, 3, 7, 9, 15, a mediana é 7, pois o número 7 é o valor do meio.

Para a sequência 1, 2, 3, 4, 5, 6, a mediana é (3 + 4) / 2 = 3.5, pois é a média dos dois valores centrais (3 e 4).

O que é Moda?

A moda é a medida de tendência central que indica o valor que ocorre com maior frequência em um conjunto de dados. Um conjunto de dados pode ter uma única moda, várias modas ou nenhuma moda, dependendo da frequência dos valores.

Fórmula: Moda = valor mais frequente.

Exemplo: Para o conjunto de dados 7, 8, 8, 9, 10, a moda é 8. Pois é o valor que aparece mais vezes.

Se você tiver o conjunto 1, 2, 2, 3, 3, 4, então os valores 2 e 3 são modas, pois ambos aparecem com a mesma frequência máxima.
